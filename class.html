Class Guide

Unofficial tier lists for PvE and PvP before and after Mental Link 1, what Mental Link changes, Elian's personal World Boss class ranking, and US server Class Card system documentation.

Source Elian (Top 50) · Discord #class Compiled Sep 2023Filed 2026

Class choice is not permanent on day one — a class change becomes available 25 days into server opening. The tier lists below use that as context: if you start with a lower-tier class, you can correct course before Mental Link content arrives and shifts the rankings.

§ 03

What is Mental Link

JP server · US equivalent is Class Cards — see § 04

Mental Link (JP server) replaces your character's Skill 3 and Rage Skill with Cygnus knight-derived skill sets. Each class gets a different set of options. The change is significant enough to shift the PvE tier list — Bowman's Mental Link rage skills tracking enemies is the specific reason Bowman jumps to S tier after this unlock.

Mental Link 1 fragments are required to unlock the feature. Mental Link 2 exists and further expands options. Video references for all skill sets were posted in the source channel but are not reproduced here due to link expiry risk — search the current game content for previews.

US server note: The US server launched its equivalent system — Class Cards — at server day 33, roughly 66 days ahead of the JP timeline. The US implementation expands the scope: all five skill slots are replaced, not just Skill 3 and the Rage Skill. See § 04 for US-specific Class Card documentation.

Earliest unlock (JP) Mental Link Auto Skill III has an earliest possible unlock at server day 102 — requiring Snow Field Common Soldier IV rank and every Frigid Snow Bones collected up to that point. This was documented in the Dojo Tips and Tricks dossier (D-11). US unlock conditions for Class Cards have not yet been cross-referenced against this JP requirement.
§ 04

Class Cards

US server · From server day 33 · Equivalent to JP Mental Link

Class Cards are the US server's implementation of what the Japanese server calls Mental Link. A Class Card overlays a sub-class skill set on top of your existing class — replacing all five auto skill slots (Auto Skills I–IV and the Rage Skill) with Cygnus Knight-derived equivalents. Your main class does not change.

JP vs US divergenceThe Japanese server did not introduce Mental Link until server day 99 (earliest unlock D102). The US server introduced Class Cards at server day 33 — approximately 66 days earlier. Additionally, the US implementation expands the replacement scope: JP Mental Link replaced Skill 3 and the Rage Skill only; Class Cards replace all five skill slots.
01Activation costs 30 fragments per card. Upgrade costs beyond initial activation are not yet confirmed.
02Three cards available at launch: Dawn Warrior, Dual Blade, Wind Archer.
03Each card replaces Auto Skills I–IV and the Rage Skill. Main class is unchanged.
04Each skill slot accepts Enhancement Cores — regular quality (D→A+) and premium quality (S→SSS).

Enhancement Cores

Each skill slot accepts one Enhancement Core that modifies how that skill performs. Cores come in two quality tracks. The regular track (D through A+) raises base DMG percentage. The premium track (S through SSS) unlocks additional effects — cooldown reduction, burst damage on proc, HP shields — and scales those effects further at SS and SSS.

Quality trackGradesCharacteristic
RegularD → C → B → A+Raises base DMG %. D tier starts at ~104%; A+ reaches ~110%. No special effects.
PremiumS → SS → SSSUnlocks special effects (cooldown reduction, burst DMG, HP shields, state bonuses). Effects scale with grade.
Core types per slot Each slot offers multiple core types to choose from. The type determines what effect the premium track unlocks — not just the magnitude. Choosing between a Mastery core (cooldown reduction) and a Soulbreaker core (raw DMG%) is the primary optimisation decision. See Dawn Warrior core data below for the full type breakdown.

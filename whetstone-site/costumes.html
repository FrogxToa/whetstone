Costume Guide

Resistance buffs by season set, the key rule that no star upgrades are needed, and which resistance types each seasonal costume covers.

Source Elian (Top 50) · Discord #costume-guide Compiled Sep 2023Filed 2026

Costumes in MapleStory R carry resistance buffs that activate on unlock — no star upgrades required. This makes costume resistance a low-friction investment: unlock the piece, get the buff. The resistance type varies by season, making it worth knowing which seasons provide which resist before spending.

CaveatSep 2023 listing. Seasons marked "?" have unconfirmed season numbers at time of writing. Verify which current season set is available and which resistance it provides before investing.
0★
Star upgrades required for resistance buffsJust unlocking the costume piece activates its resistance stat. You do not need to upgrade to any star level. Unnecessary star spending wastes resources.

§ 02

Which resist to prioritise

Content-dependent

Hex Time Reduction

Reduces the duration of hex/rage-lock debuffs. Relevant in Dojo hex-locked stages and PvP. Prom set is the source — cross-references the Dojo Tips dossier (D-11).

Stun Resistance

Reduces stun duration. Hawaiian and British sets both provide this. Relevant across PvP and many boss encounters. Two sets available for stacking.

Petrify Resistance

Reduces petrify duration. Exoskeleton (Season 1) is the source. Cross-references the Dyle pet (petrify resistance) and EXR Soul Teddy (reduces enemy petrify resistance).

Stacking armour + hairEach season provides both an armour and a hair piece. Unlocking both from the same set gives 7.5% total resistance (5% + 2.5%). Stacking sets from different seasons with the same resistance type (e.g. Hawaiian + British for stun) gives 15% stun resistance across armour + hair from both sets.

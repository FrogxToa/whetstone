Class Guide

Unofficial tier lists for PvE and PvP before and after Mental Link 1, what Mental Link changes, and Elian's personal World Boss class ranking.

Source Elian (Top 50) · Discord #class Compiled Sep 2023Filed 2026

Class choice is not permanent on day one — a class change becomes available 25 days into server opening. The tier lists below use that as context: if you start with a lower-tier class, you can correct course before Mental Link content arrives and shifts the rankings.

CaveatOne player's opinion from Sep 2023. Explicitly stated as unofficial and lacking sufficient data for Mage and Warrior at the time of writing. Treat as a directional reference, not a definitive ranking.

§ 03

What is Mental Link

Explained

Mental Link replaces your character's Skill 3 and Rage Skill with Cygnus knight-derived skill sets. Each class gets a different set of options. The change is significant enough to shift the PvE tier list — Bowman's Mental Link rage skills tracking enemies is the specific reason Bowman jumps to S tier after this unlock.

Mental Link 1 fragments are required to unlock the feature. Mental Link 2 exists and further expands options. Video references for all skill sets were posted in the source channel but are not reproduced here due to link expiry risk — search the current game content for previews.

Earliest unlock Mental Link Auto Skill III has an earliest possible unlock at server day 102 — requiring Snow Field Common Soldier IV rank and every Frigid Snow Bones collected up to that point. This was documented in the Dojo Tips and Tricks dossier (D-11).

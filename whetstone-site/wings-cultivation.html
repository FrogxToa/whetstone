Wings Cultivation Events

CD spend per Nirvana level, fragment point values, Soul Power of Fire upgrade cost table, and the prerequisite to unlock the Soul Power system.

Source Elian (Top 50), Firefighter Ulam · Discord #wings-cultivation-event Nirvana calc credits @Juicyaf Compiled Sep 2023Filed 2026

Wings cultivation events run for 7 days and reward wing fragments based on points earned from submitting wing-type items. The Nirvana levels are the point milestone tiers — each requires a specific CD investment if you are purchasing event packs. The Soul Power of Fire system is a separate permanent upgrade that uses wing fragments.

CaveatSep 2023 data. Fragment point values, pack contents, and Nirvana thresholds may have changed. Use as a planning framework, not exact current values.
